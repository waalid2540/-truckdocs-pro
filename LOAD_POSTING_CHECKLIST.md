## Speed Tips

1. **Tab key is your friend** - Don't use mouse between fields
2. **Skip optional fields** - Only fill if readily visible on DAT
3. **Round numbers** - $2,487 ‚Üí Enter as `2487` (no commas or $)
4. **Date format** - Use date picker or type YYYY-MM-DD
5. **Don't overthink** - Imperfect load posted > perfect load not posted

## Troubleshooting

**"Load already exists" error:**
- Someone else posted this load (you or another user)
- Skip it and move to next load

**"Missing required field" error:**
- Check all 7 required fields are filled
- Make sure Total Rate is a number (no $ or commas)
- Make sure date is in correct format

**"Failed to post load" error:**
- Check your internet connection
- Try refreshing the page
- Clear browser cache
